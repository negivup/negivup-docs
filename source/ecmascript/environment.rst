基础环境
================================

Node 版本
--------------------------------

保证电脑中安装了 Node.js。

.. code-block:: bash

    $ node -v
    v11.10.1

    $ npm -v
    6.14.4

    $ npx -v
    6.14.4

注意：这里列出的是最低要求的软件版本，在实际学习过程中最好和列出的版本保持一致。

学习环境
--------------------------------

浏览器中并不一定能够 100% 支持 ECMAScript 的所有新特性，因此需要搭建一个基础的学习环境，用于学习最新的语法。
