基础环境
================================

Node 版本
--------------------------------

保证电脑中安装了 Node.js。

.. code-block:: bash

    $ node -v
    v11.10.1

    $ npm -v
    6.14.4

    $ npx -v
    6.14.4

注意：这里列出的是最低要求的软件版本，在实际学习过程中最好和列出的版本保持一致。
